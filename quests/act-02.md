# 第二幕剧本

> **格式说明**（同第一幕）
> - 每一行 = 一个对话框
> - `[旁白]` = 旁白框
> - `[内心]` = 主角内心独白
> - `[选择]` = 玩家选项
> - `[判定]` = 条件检测
> - `[获得道具]` = 道具取得提示
> - `[旗标]` = 设置或检测游戏内部旗标
